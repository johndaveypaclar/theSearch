Search Engine Optimization and Search Engine Marketing

by DAVE

Summary

Search Engine Marketing or SEM is a form of Internet marketing that involves promotion of websites by making the website higher in the search engine results.

  • Domain name Strategy
  • Linking Strategy
  • Keyword Strategy
  • Title Tags Strategy
  • Meta Description Tag Strategy
  • Alt Tags Strategy
  • Search Engine Indexing Strategy

From my previous blog, I talked about the simplest thing that you should need to know about SEO. We already know that SEO or Search Engine Optimization is a methodology of techniques and strategies that will make your website reach the top in search engines. Now, I’m going to introduce SEM or Search Engine Marketing. The SEM is the marketing process with a goal of getting more visibility in search engines (Google, Yahoo, Bing) either by getting more free traffic or paid traffic.

There are 7 Strategies in SEO:

>Domain name Strategy. A strategy that focuses on making a unique and simple way of naming the domain name. The domain names are like this; facebook.com, twitter.com, etc.

>Linking Strategy. Links have the strongest association for the highest search rankings for all the attributes used by Google in its algorithm. This is one of the problem of small businesses, is a struggle to build quality links.

>Keyword Strategy. Keywords are crucial in developing winning search engine marketing campaigns. The keywords should involve high-performing keywords or keywords that drive relevant traffic to the business. Choosing the best keyword can make a huge difference in your campaigns.

>Title Tags Strategy. Title tags is the thing that makes people decide whether they will visit the website or not. The title tags should have important keywords to help the search engine determine what the page is about.

>Meta Description Tag Strategy. This serves the function of advertising copy. It draws the readers to a certain website from the SERP, and this is a very visible and one of the most important part of search marketing. Creating a readable or easy to understand, compelling description using important keywords can improve the rate of the page.

>Alt Tags Strategy. An HTML code that describes the image that appears on the web page so that every user will know what the image represents if they really can’t see properly. This will not help the site climb the search results, but this will improve the SEO performance.

>Search Engine Indexing Strategy. This is a process of search engine collecting, parses and stores data for use by the used search engine. The actual search engine index is the place where every data the search engine has already collected and is stored.

Now thinking on what strategy to use? It is possible to combine strategies. The most of the famous marketing websites really think about the strategies that they use and it also depends on what the client or the business needs.
